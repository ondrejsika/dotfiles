My configuration files for desktop (Mac) and servers (Debian)


## Desktop (Mac)

```
./apply-mac.sh
```

### Install Brew Packages

```
./brew-install.sh
```

### Save Brew Packages

```
./brew-save.sh
```

## Server (Debian)

```
./apply-server.sh
```
